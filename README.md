# About

A simple todo-list web app made using Ejs, NodeJS, ExpressJS, and MongoDB as the database as a practice project.

## Requirements

-NodeJS environment

-MongoDB

## Usage
 Just fork the project and feel free to do any kind of improvements.
```python
npm install
node app.js
```
